#おやすみモード(Do Not Disturb)

Androidでおやすみモードを実現するためのアプリです。

##インストール方法

1. Android端末で[こちら](./app/app-release.apk)からAPKをダウンロードする

2. 「設定 -> セキュリティ -> 提供元不明のアプリ」にチェックを入れる

3. 「ダウンロード -> app-release.apk」からインストールする

##使い方

1. 開始・終了時刻を設定してから、「有効にする」をタップする
